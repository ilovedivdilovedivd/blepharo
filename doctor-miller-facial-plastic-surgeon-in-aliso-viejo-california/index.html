Would a facelift help with bells palsy?
Would a facelift help with bells palsy?
Aug 9, 2017
In general, episodes of Bell's palsy resolve.  However, in some cases the return of facial function may take time (years) and/or may not be 100% complete.  Using surgery such as a facelift can be used to correct asymmetries but is typically reserved when it is determined that return of nerve function is highly unlikely.  The recommendation should be done in concert with the physician (typically an otolaryngologist) managing your Bell's palsy.I hope this helps.
